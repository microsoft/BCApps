# Tone and Style Analysis

Analyze the tone and style of a document with examples and effectiveness assessment.

## When to Use

- Evaluating writing for audience fit
- Improving document consistency
- User asks about "tone", "style", "voice", or "how does this read"

## Analysis Dimensions

### Tone Categories

| Tone | Characteristics |
|------|-----------------|
| **Formal** | Professional language, third person, complete sentences |
| **Informal** | Casual language, contractions, conversational |
| **Persuasive** | Action-oriented, emotional appeals, calls to action |
| **Informative** | Objective, factual, educational |
| **Authoritative** | Confident, definitive statements, expert voice |
| **Empathetic** | Understanding, supportive, reader-focused |
| **Urgent** | Time pressure, immediate action needed |
| **Cautious** | Hedged language, qualifications, disclaimers |

### Style Categories

| Style | Characteristics |
|-------|-----------------|
| **Academic** | Citations, formal structure, technical vocabulary |
| **Conversational** | Direct address, questions, everyday language |
| **Technical** | Precise terminology, detailed specifications |
| **Journalistic** | Inverted pyramid, quotes, attribution |
| **Marketing** | Benefits-focused, emotional triggers, CTAs |
| **Legal** | Precise definitions, comprehensive coverage |
| **Narrative** | Storytelling, chronological, character-driven |

## Tone Indicators to Watch For

| Indicator | Examples |
|-----------|----------|
| **Word choice** | Technical vs. simple, active vs. passive |
| **Sentence structure** | Short/punchy vs. long/complex |
| **Person** | First (I/we), Second (you), Third (it/they) |
| **Qualifiers** | "Definitely" vs. "perhaps" vs. "might" |
| **Emotional words** | "Exciting" vs. "notable" vs. "concerning" |
| **Questions** | Rhetorical, direct, or none |
| **Contractions** | "Don't" vs. "do not" |
